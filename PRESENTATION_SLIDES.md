## สไลด์ที่ 4: หลักการทางวิทยาศาสตร์

### Beer-Lambert Law

```
A = ε × c × l

A = Absorbance (ความดูดกลืนแสง)
ε = Molar absorptivity
c = Concentration (ความเข้มข้น) ← สิ่งที่เราต้องการหา
l = Path length
```

### การประยุกต์ใช้

```
ความเข้มของสี ∝ ความเข้มข้นของสาร
      ↓
ค่า RGB ← วัดจากภาพ
      ↓
Machine Learning เรียนรู้ความสัมพันธ์
      ↓
ทำนายความเข้มข้น
```
